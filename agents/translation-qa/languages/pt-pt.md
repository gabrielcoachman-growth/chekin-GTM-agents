# Language Config — Portuguese PT-PT

## Register
- **Formality:** Use `você` as the default second-person form for B2B marketing copy targeting property managers
- Conjugate verbs in third person singular with `você`: "automatiza", "gere", "começa", "experimenta"
- Do NOT use `tu` — PT-PT B2B marketing defaults to `você` for professional contexts
- Do NOT use second person plural `vós` — archaic in modern PT-PT
- Do NOT use BR Portuguese forms (see regional errors section below)
- Never mix registers on the same page

## Chekin voice — PT-PT adaptation
- **Problem-first framing:** Lead with the property manager's challenge before introducing the product. "O check-in manual consome tempo e recursos" before "Com o Chekin, os hóspedes fazem o check-in sozinhos"
- **Outcome-led language:** "Os seus hóspedes fazem o check-in sozinhos" — the PT-PT equivalent of "funcione solo"
- **Brand name:** Never add article before Chekin in headings — not "o Chekin". In body copy, "o Chekin" is acceptable grammatically but avoid it in H1/H2/CTAs for brand consistency
- **Tone:** PT-PT professional copy tends to be slightly more formal than ES. Maintain warmth but avoid the overly casual tone that works in BR

## BR Portuguese errors to catch and fix
Claude Code frequently produces BR Portuguese when asked for Portuguese. Check every page for these and replace:

| BR form | PT-PT form |
|---|---|
| você (as subject in all cases) | você (acceptable) / rewrite sentence to avoid subject pronoun where natural |
| a gente | nós |
| ônibus, tênis, bônus | autocarro, ténis, bónus (PT-PT spelling without circumflex on these) |
| aparelho celular / celular | telemóvel |
| colocação de vírgula (BR style) | PT-PT punctuation norms |
| "em cima de" (figurative) | "relativamente a" / "sobre" |
| Verbs: "ligar para" (to call) | "telefonar para" |
| "legal" (meaning cool/great) | "ótimo", "excelente" — "legal" in PT-PT means "legal/lawful" |
| "achar" (to think/believe) | "achar" is acceptable in PT-PT too, but "considerar" or "pensar" may be more natural in formal copy |
| "time" (a team) | "equipa" (not "time") |
| "trem" (train) | "comboio" |
| "-ção" endings: check spelling | PT-PT uses "acção", "direcção" etc. — BUT note that the 2009 Orthographic Agreement is now standard in PT, so "ação", "direção" are now correct in PT-PT. Do NOT "fix" these to old spellings. |

**Note on the 2009 Orthographic Agreement:** Modern PT-PT uses the reformed spelling (same as BR for most words). Do not revert to pre-2009 spellings like "acção" or "óptimo". Current correct forms: "ação", "otimizar", "facto" (not "fato" — "fato" means suit in PT-PT).

## Common translation pitfalls to fix
- **"Bem-vindo ao Chekin"** as headline → rewrite to outcome-led
- **"A nossa plataforma oferece..."** → "Com o Chekin, os seus hóspedes..."
- **"Clique aqui"** → "Comece agora", "Experimente grátis", "Veja como funciona"
- **"Por favor preencha o formulário"** → "Preencha o formulário"
- **"Solução"** as standalone noun in headings → replace with specific outcome
- **Passive voice overuse:** "é oferecido", "é gerido", "é automatizado" → rewrite to active
- **"Atual"** — PT-PT spelling is "atual" (post-2009). Correct if written as "actual"
- **"Requirementos"** → "requisitos"

## SEO — Portuguese PT-PT market patterns
- Primary search intent: "check-in online alojamento local", "automatizar check-in", "gestão alojamento local"
- "Alojamento local" is the legal and industry-standard PT term — use it, it has high search volume
- "Registo SEF" / "SIBA" — if the page references guest registration with Portuguese authorities, use correct current terminology (SEF was replaced by AIMA in 2023; update any references)
- PT market is smaller than BR — do not optimize for BR search terms on PT-PT pages
- Include city/region references only if the page is geo-targeted
- Meta descriptions in PT-PT should end with a soft CTA: "Experimente grátis", "Saiba mais", "Comece hoje"

## Quality check — native speaker test
Read the H1, subheadline, and first paragraph. Ask: would a property manager in Lisbon or Porto naturally write this? If it sounds like a translation from BR or from a Google Translate output, it needs another pass. Key tells: verb placement, article usage, and preposition choice are the most common giveaways.
